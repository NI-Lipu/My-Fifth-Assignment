Important FAQ

What is the Document Object Model(DOM)?

The full meaning of DOM is Document Object Model.We can access and modify any kind of HTML structure and CSS style by using Javascript DOM.

How do you select an element from the DOM?

There are several way to select an element from the DOM.
They are--
# getElementsByTagName()
# getElementsByClassName()
# getElementById()
# querySelector()
# querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

By event delegation we can handles all child elements by a single parent element. It's useful because it improves performance, reduces memory usages and simplifies the code.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles using the DOM.
To manipulate attributes we can use --
# getAttribute()
# setAttribute()
# removeAttribute()

To manipulate styles we can use --
classList.add()
classList.remove()
classList.toggle()
classList.contains()
